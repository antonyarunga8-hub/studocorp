## 🚀 Deployment

### Netlify (recommended)
1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → **Add new site → Import from Git**
3. Select your repo — Netlify auto-detects the config
4. Click **Deploy site** ✅

### Vercel
1. Go to [vercel.com](https://vercel.com) → **New Project → Import Git Repository**
2. Select this repo — no build config needed
3. Click **Deploy** ✅

### GitHub Pages
1. Go to your repo → **Settings → Pages**
2. Set source to `main` branch, `/ (root)` folder
3. Save — site live at `https://yourusername.github.io/studocorp/`
